# DCMLockerLast                                                                                                      

Build-artifact repository for the Raspberry-Pi locker line. Contains the compiled output of [`DCMSolutions/DCMLockerCliente`](https://github.com/DCMSolutions/DCMLockerCliente) — the published build of the `DCMLocker.sln` Blazor solution.                                                                                     
  
This repository holds **binaries, not source**. Source code, build instructions, and any reproduction or modification of these artifacts are governed by `DCMSolutions/DCMLockerCliente`.

## Status

No active development. The repository is preserved for continuity and may receive occasional maintenance commits.
